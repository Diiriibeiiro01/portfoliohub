# PortfolioHUB — Mateus Vinicius

Plataforma centralizada para exibir e gerenciar projetos e portfólios digitais.

## 🔗 Acesse
> https://seuusuario.github.io/portfoliohub

## 🛠️ Tecnologias
- HTML5
- CSS3 (responsivo, sem frameworks)
- JavaScript vanilla
- GitHub Pages (hospedagem)

## 📁 Estrutura
```
portfoliohub/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md
```

## 🚀 Como rodar localmente
```bash
git clone https://github.com/seuusuario/portfoliohub.git
cd portfoliohub
# Abra o index.html no navegador
```

## 📌 Como adicionar um projeto
Edite o `index.html` e copie um bloco `.proj-card` dentro de `.proj-grid`.
